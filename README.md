# FakeBookGraphQL

FakeBook GraphQL API

## Model 

- Book
    - Has many songs
    - Has a reference to the whole pdf

- Song:
    - Page in book
    - Title
    - Reference to the associated pdf
    - song start page
    - song end page

## Capabilities

- Search song by title or page number
- Browse books
